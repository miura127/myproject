# Bookmark Library
　参考になった技術記事を管理するサイト
　
　※未デプロイ

## 使用技術
- 言語
    - Java 8
- フレームワーク
    - SpringBoot
    - Mybatis
    - Lombok
    - Spring Security
    - Spring AOP
    - MapStruct
    - Bootstrap
- テンプレートエンジン 
    - thymeleaf
- ビルド管理ツール
    - Maven
- データベース
    - H2 Database → PostgreSQLに変更(2021/04/11)
- IDE
    - eclipse

## 実装済の機能
- 投稿一覧表示
- 投稿詳細表示
- 新規投稿
- 投稿編集
- 投稿削除
- ユーザー登録
- ログイン/ログアウト
- ロールを用いたアクセス制御
- カテゴリ検索
- キーワードによるあいまい検索
- 件数取得
- AOPによるログ出力
- ログインユーザー情報取得(「誰が投稿したか」をDBに登録)

## これから実装したい機能
- 投稿一覧のURLをリンクとして表示する
- ページネーション
